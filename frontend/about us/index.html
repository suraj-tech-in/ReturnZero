About Us

Innovathon is a dynamic platform that centralizes all tech events and hackathons happening across Nepal, creating a unified space for discovery and participation. It simplifies the process of finding events, providing detailed information about upcoming opportunities in the tech community. With features like seamless team registration and an intuitive user interface, Innovathon caters to both event organizers and participants, fostering collaboration and innovation. By connecting tech enthusiasts nationwide, the platform aims to build a thriving ecosystem where ideas can flourish and talents can shine.
